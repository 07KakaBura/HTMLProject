What is HTML and the purpose of it

What is HTML?

  1. Definition:
    • HTML stands for:
      1. Hypertext
      2. Markup
      3. Language
  2. Standard Language:
    • HTML is the standard markup language for creating web pages. Example of some of the oldest websites:
      1. World Wide Web
      2. First Website Using Line Mode Browser Simulator
  3. Structure:
    • It provides the structure or skeleton of web pages.
      • Learn More about this here.
        1. Web Page skeleton
  4. Markup Language:
    • HTML uses markup tags to define elements within a document example:
      1. Headings and sub headings:
        • <h1>, <h2>, <h3>, <h4>,<h5>
      2. For body/paragraphs:
        • <body>
      3. lists
        1. Ordered List:
          • <ol><li>content</li></ol>
        2. Unordered List:
          • <ul><li>content</li></ul>
  5. Text Formatting:
    • HTML allows formatting of text, such as headings, paragraphs, and lists.
      1. Headings and sub headings:
        • Search engines use the headings to index the structure and content of your web pages.
          • <h1>, <h2>, <h3>, <h4>,<h5>
      2. For body/paragraphs:
        • The <body> tag defines the document's body.
          • <body>Document Content</body>
      3. Lists:
        • An unordered list starts with the <ul> tag. Each list item starts with the <li> tag. The list items will be marked with bullets (small black circles) by default.An ordered list starts with the <ol> tag. Each list item starts with the <li> tag. The list items will be marked with bullets (small black circles) by default.
          1. Ordered List:
            • <ol><li>content</li></ol>
          2. Unordered List:
            • <ul><li>content</li></ul>
  6. Hyperlinks:
    • It enables the creation of hyperlinks to link different web pages and resources.
      • Hyperlinks are represented by <a>
        1. Example <a href="link"> Text to link </a >
  7. Images:
    • HTML facilitates the insertion and display of images within web pages.
      • Images are represented by <img>
  8. Forms:
    • It supports the creation of interactive forms for user input.
      • Forms are represented by < form >
  9. Tables:
    • HTML allows the creation of tables to organize and present data.
      • It is represented by < table >
  10. Semantic Markup:
    • HTML5 introduced semantic elements for better structure and semantics.

What is the purpose of html?

  • Structuring Content:
    • HTML provides a structural framework for organizing content on web pages.
      • <div>, <p>, <header>, <footer>
  • Semantic Markup:
    • It enables the use of semantic elements to give meaning to different parts of a web page, aiding accessibility and search engine optimization (SEO)
  • Content Presentation:
    • HTML facilitates the presentation of text, images, videos, and other multimedia elements on web pages.
      • Organizing Lists:
        • <ul>, <ol>,<li>
  • Hyperlink Creation: <img>, <audi, <video>
